## 🔎 HEG-2の概要──ことばは関係の振動である

HEG-2（RL: 関係性言語論）は、以下のような多層構文から成る：

- **構文（syntax）** = 意味の種
    
- **意味（semantics）** = 振動としての再配置
    
- **語用（pragmatics）** = 関係の中で揺れる意図と実践
    
- **詠（poetics）** = ふちに触れたとき、意味の外側で発生する拍
    

この理論が提示するのは、「言語とは何か」ではなく、  
**言語がどのように誰かに響き、生きるのか**である。
