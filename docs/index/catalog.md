---

## 🔁 更新フロー
1. 文章やデータを追加 → `articles/` or `STPS/manifesto/` 等に配置  
2. この目録に行を追加し、EgQE-XXX / STPS-XXX 番号を採番  
3. Release作成 → Zenodo自動連携 → DOI発行  
4. DOIを本テーブルに追記  
5. 外部記事（note/Mediumなど）で DOI を引用
