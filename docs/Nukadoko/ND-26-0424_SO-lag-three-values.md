# SO lag（三値）

## ── 揃わなさの三つの様式

---

lagとは、揃わなさである。

---

```text
rate_s ≒ rate_o
rate_s > rate_o
rate_s < rate_o
```

---

```text
≒ → 持続（sustain）
>  → 展開（expansion）
<  → 保持（retention）
```

---

揃えば持続し、 ずれれば流れ、 逆転すれば引かれる。

---

SO lag は、rate_s と rate_o の関係が揃わないことである。

---

完全一致は存在しない。

---

揃わないから、世界は持続する。
