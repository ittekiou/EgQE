# ぬか床｜時間とは何か

## Time as the Irreversible Accumulation of Lag

_漬け込み日：2026-04-25_

---

## 軸

**時間 = lagの蓄積が不可逆に残ること**

```
時間 ≠ 流れ
時間 ＝ 残り
```

**時間とは、消えきれなかった差異が積もり続けることである**

---

## 素材（EXシリーズから）

```
EX-01：生成（場の成立）
EX-02：持続（B）
EX-03：分化（Z₀）
EX-04：崩壊（地層）
```

👉 これそのまま時間の1周期

---

## 構造

```
生成 → 持続 → 分化 → 崩壊
              ↓
           ΔZ蓄積
              ↓
           次の生成
```

**循環しているが、同一には復帰しない。**

```
同じ場所に戻る　→ ○
同じ状態に戻る　→ ×
```

👉 ここが時間。

---

```text
不可逆とは、ΔZが消去されないことである。
```

```text
Irreversibility means that ΔZ cannot be erased.
```
