# while と as の位相差（seed）

── 一語が切断を露出する

---

## 入口

while it strolls, it seeds. as it strolls, it seeds.

この差は文法ではない。 位相差そのものである。

---

## 候補行（寝かせ中）

切断はwhileでは起きない。 asでしか起きない。

---

## 構文メモ

```
while = 並行（二つの行為）
as    = 同一運動（分離不可能）
```

whileは並びである。 asは同一運動である。

asとは、切断がすでに内在している位相である。

---

## 位置づけ

```
while it strolls, it seeds.
→ stroll + seed（合成位相）

as it strolls, it seeds.
→ stroll = seed（内在位相）
```

Z₀的に言うと—— 切断は行為ではない。 運動の中で起きている。
