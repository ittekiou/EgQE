## 「光速定数は排泄だった」問題

直感：当たり。表現：一段ずらす必要あり。

```
排泄（Z₀）が直接定数になるわけではない。
排泄 = 切断 → その痕跡がΔZ（定数）になる。
```

> 光速定数は、排泄の痕跡である。

> 排泄は光速的であり、光速は排泄の条件である。

> 排泄が走り、痕跡が光速になる。
