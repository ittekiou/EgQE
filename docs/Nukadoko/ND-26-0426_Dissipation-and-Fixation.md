# ぬか床｜散逸と固着——非folding問題の統合

## Dissipation and Fixation as Two Modes of Non-Folding

_漬け込み日：2026-04-26_

---

## 問い

v5.1：非folding → 散逸  
EX-04：原油 → 固着

一見逆。しかし構文で見ると統一できる。

---

## 結論

**散逸と固着は、再帰に入らなかったlagの二つの様式である。**

---

## 構造

```
lagが再帰に入らない
↓
内部保持されない
↓
Z₀へ行かない
↓
分岐：

  分散 → 散逸（低密度状態）
  局在 → 固着（高密度状態）
```

---

## B（境界強度）で見ると三相

```
低B  → 散逸
高B  → 固着
臨界B → Z₀（foldingが成立する点）
```

---

## 決定的再定義

```
非folding = 持続へ移行しないlag

散逸 = 再帰に入らず拡散したlagの低密度状態
固着 = 再帰に入らず局在したlagの高密度状態

原油 = 再帰に入らず固着したlagの高密度状態
```

---

## Z₀の位置

> **Z₀は散逸でも固着でもない。**  
> **散逸と固着のあいだで、foldingが成立する臨界点である。**

---

## v5.1とEX-04の統合

```
v5.1：非folding → 散逸側を記述
EX-04：非folding → 固着側を記述

同じ構造の別モード。
```

---

## 一行

```
非foldingは散逸でも固着でもない。

散逸はその低密度状態、
固着はその高密度状態である。
Z₀はそのあいだで成立する。
```
