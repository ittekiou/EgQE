---

## 0. 更新の要点

Z₀はもはや「単位」ではない。  
また単なる「切断」でもない。

**Z₀は、場の中で成立する臨界である。**

---

## 1. 問題設定｜非foldingの分岐

従来：

```
非folding → 散逸
```

しかし：

```
非folding → 固着（原油など）
```

**一見逆の現象が存在する。**

---

## 2. 再定義｜非folding

> **非foldingとは、再帰（持続）へ移行しないlagである。**

---

## 3. 二様式｜散逸と固着

非foldingは二つの様式を取る：

```
低密度 → 散逸（拡散）
高密度 → 固着（局在）
```

> **散逸と固着は、再帰に入らなかったlagの二つの様式である。**

---

## 4. 三相構造｜B（境界強度）

境界強度 B によって全体は三相に分かれる：

```
低B   → 散逸
臨界B → Z₀（folding成立）
高B   → 固着
```

---

## 5. Z₀の再定義

> **Z₀とは、散逸と固着のあいだでfoldingが成立する臨界である。**

これは：

- 状態ではない
- 単位ではない

**👉 位相境界である**

---

## 5.1 Z₀の非対称性

Z₀は不可逆である。  
一度成立したfoldingは、同一条件には戻らない。

> **散逸と固着は対称だが、Z₀は非対称である。**

Z₀だけが時間を生む。
