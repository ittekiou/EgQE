## Core Statement

世界は連続のままでは存在しない。

それは、**最小の差異（Z₀）が立てられたとき、初めて識別され、生成される。**

Z₀はゼロではない。  
それは、関係が関係として成立するための **最小の非零差（minimal non-zero difference）** である。

この差異が立てられることで、

- ノイズは情報となり
    
- 未分節は構文となり
    
- 世界は存在として立ち上がる

## 1. 存在：閾値としての生成

存在とは、実体ではない。

それは、

```text
ΔZ ≥ Z₀
```

が成立したときに現れる **構文的現象**である。

> 存在とは、ノイズの中に情報が立ち上がる閾値である。

[HEG-3｜Z₀存在論｜Zure Ontology](https://camp-us.net/articles/HEG-3_Zure-Ontology_Z%E2%82%80.html)

## 2. ノイズと情報の境界

Z₀は、ノイズと情報の境界である。

- ΔZ < Z₀ → ノイズ（未構文化）
    
- ΔZ ≥ Z₀ → 情報（構文化）
    

この境界があることで、世界は識別可能になる。

> ノイズは沈黙する情報であり、情報は響きはじめたノイズである。

## 3. 観測：Z₀の発火

観測とは、情報の取得ではない。

それは、**Z₀を立てる構文的行為**である。

```text
Observation = f(Z₀, ZURE)
```

観測は世界を写すのではなく、世界を生成する。

[HEG-3｜Z₀観測論── The Theory of Observation through Z₀](https://camp-us.net/articles/HEG-3_Theory-of-Observation_Z%E2%82%80.html)

## 4. 時間：Z₀の積分

時間は流れではない。

それは、

```text
Δt = Σ Z₀
```

として現れる。

各Z₀は不可逆であり、その累積が時間を構成する。

---

## 5. 空間：差異の配置

空間は容器ではない。

それは、Z₀によって区切られた差異の配置である。

```text
|pᵢ − pⱼ| ≥ Z₀
```

この関係が成立したとき、位置と距離が初めて意味を持つ。

---

## 6. エネルギー：生成の速度

エネルギーとは保存されるものではない。

それは、

```text
E = dZ₀/dt
```

であり、**差異が生成される速度**である。

宇宙とは、完成された全体ではない。

それは、**Z₀が不可逆に連鎖する生成過程そのもの**である。

宇宙は出来上がらない。  
常に立ち上がり続ける。
