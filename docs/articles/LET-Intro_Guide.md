# lag-edit theory｜入口ガイド
## **── どこから読んでも、同じ回路に合流する**

---

lag-edit theory は、ひとつの説明では入れない。  
どこから入るかで、見え方が変わる。

「あ、わかった」が起きる場所が違う。  
人によって、直感のスイッチが違う。

だから入口を複数用意した。

どこから入っても、最後は同じ場所に着く。
