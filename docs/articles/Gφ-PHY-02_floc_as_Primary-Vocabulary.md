## Abstract

本稿は、物理学の中心概念——エントロピー・エネルギー・非平衡——を floc・lag・φの文法で読み直す。

これは既存物理学への反論ではない。 **別の語彙で同じ宇宙を記述する試みである。**

---

## 1. 既存物理学の語彙

物理学はこれまで、二つの中心概念で宇宙を記述してきた。

```
エントロピー　→　無秩序への傾向
エネルギー　　→　秩序の担体
```

この枠組みでは：

- 宇宙はエントロピーが増大する方向に進む
- エネルギーが秩序を維持する

---

## 2. flocで読み直す

Cosmogonica の語彙で同じ宇宙を見ると：

```
エントロピー　→　floc（非局所的揺らぎの充満）
エネルギー　　→　lag（差異の拘束様式）
非平衡　　　　→　φ（更新を伴う持続）
```

エントロピーは「無秩序」ではなく、 **遭遇以前の充満した静止**（[SN-DK-01](https://camp-us.net/articles/SN-DK-01_Darkness-Stillness-Hypothesis.html)）。

エネルギーは「力」ではなく、 **lagがどのように拘束されるかの様式**（[SN-ORDER-01](https://camp-us.net/articles/SN-ORD-01_Typology-of-Order_as_Lag-Modes.html)）。

非平衡は「乱れ」ではなく、 **φにおける更新の持続**（[5 Recursive](https://camp-us.net/articles/SN-ORD-01_Typology-of-Order_as_Lag-Modes.html)）。

---

## 3. 語彙の差異

二つの語彙は、同じ現象を異なる方向から記述する。

```
物理学語彙　　　　　Cosmogonica語彙

エントロピー　←→　floc
エネルギー　　←→　lag
非平衡　　　　←→　φ更新持続
無秩序　　　　←→　遭遇以前の充満
秩序崩壊　　　←→　Crystal終端
```

物理学は**結果**を記述する。 Cosmogonica は**生成の過程**を記述する。

---

## 4. flocという語彙

なぜflocか。

エントロピーは増大を語る。 エネルギーは保存を語る。

flocは**生成を語る。**

```
floc
↓
lag（差異の拘束）
↓
φ（分岐）
↓
秩序の多様性（0-6類型）
```

この系列は、エントロピーでもエネルギーでも 記述できなかった**生成の文法**を持つ。

---

## 5. 結論

> floc is the primary generative vocabulary.  

flocこそが生成を記述する基礎語彙である。

エントロピーは宇宙の終わりを語る。 エネルギーは宇宙の均衡を語る。

flocは**宇宙の始まりを、始まりなしに語る。**
