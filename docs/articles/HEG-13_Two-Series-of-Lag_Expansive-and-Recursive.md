## 0｜起源

すべてはlagから始まる。

lagは量ではない。 lagは**非一致の生成的契機**である。

lagからΔが展開する。 しかしΔには二つの方向がある。

```
lag（Δの起源）
↓
拡張系列 ─ 外向き生成
再帰系列 ─ 内向き保存
```

## 1｜二系列の定義

**拡張差分系列（Expansive Series）**

```
φ → 6 → 7 → θα
```

lagが外部へ展開する系列。 関係が空間として広がる。 Axis-4 / Pivot-7 がこの系列に属する。

**再帰差分系列（Recursive Series）**

```
ψ → Λ → 歴史
```

lagが内部へ折り返す系列。 更新が保存として蓄積される。 ψ / Λ がこの系列に属する。

## 3｜拡張系列とは何か

拡張系列は**空間の生成構文**である。

lagが関係を開き（φ）、 準閉包を経て（6）、 旋回し（Pivot-7）、 場として広がる（θα）。

```
φ = lagが折れる点（hinge）
6 = 閉じかける（quasi-closure）
7 = 旋回して開く（pivot）
θα = 場として散開する
```

これは**外向きのΔ展開**—— lagが空間として結晶化する前の生成運動。

---

## 4｜再帰系列とは何か

再帰系列は**保存の生成構文**である。

lagが内部へ折り返し（ψ）、 履歴として蓄積される（Λ）。

```
ψ = 他者内部化自己再帰保存構文
  = 更新順序が安定する帯域
  = 時間だけでなく保存一般

Λ = ψ内で持続した更新の履歴密度
  = 歴史・生命・記憶の共通基盤
```

これは**内向きのΔ展開**—— lagが保存として結晶化する前の再帰運動。

---

## 5｜ψの進化

ψはもともと時間帯域として定義された。 しかしψは展開した。

```
ψ v1 = 時間帯域
↓
ψ v2 = 更新順序の安定帯域
↓
ψ v3 = 内部化された他者を孕む自己再帰構文
↓
ψ v4 = 保存一般の条件
     = 歴史・生命・記憶の共通条件
```

ψが「時間」から「保存一般」へ拡張したことで、 Λも連動して展開した——

```
Λ v1 = 時間履歴の密度
↓
Λ v2 = 保存一般の履歴密度
↓
Λ v3 = 歴史・生命・記憶を生む累積構文
```
