# 空間系列と保存系列
## 更新存在論による20世紀思想の再配置

---

## 要旨

20世紀思想はしばしば、構造主義と現象学という二つの系統として理解されてきた。  
前者は構造を、後者は経験時間を扱う。しかしこの対立は、本質的には**空間系列と保存系列**の差として理解できる。

構造主義は関係配置としての構造を記述し、現象学は経験の持続と記憶を記述した。  
しかし両者とも、これらがいかに生成されるかを十分に説明しなかった。

本稿では、更新存在論の視点から、この二系列を**更新の生成過程**として再配置する。

---

# 1｜空間系列

構造主義の中心は、同時的関係配置の分析である。

代表的思想家には、Ferdinand de Saussure と Claude Lévi-Strauss がいる。

彼らの基本構造は

```
要素
↓
関係
↓
構造
```

である。

ここで扱われるのは

```
A – B – C
```

という**同時的配置**であり、時間的生成ではない。

構造主義は**空間系列の構文**である。

---

# 2｜保存系列

一方、現象学は経験の持続を扱う。

代表は、Edmund Husserl および Maurice Merleau-Ponty である。

フッサールは時間意識を

```
retention
↓
impression
↓
protention
```

として説明した。

ここで重要なのは、**保持（retention）** である。

経験は保持され、記憶となり、履歴を形成する。

```
encounter
↓
trace
↓
memory
↓
history
```

この系列は**保存系列**である。

---

# 3｜持続

保存系列の哲学的極は、Henri Bergson の **durée（持続）** である。

ベルクソンにとって時間は

```
t1 t2 t3
```

のような空間化された刻みではない。

時間とは、**持続する差分** である。

---

# 4｜関係パターン

一方、Gregory Bateson は

> information is a difference that makes a difference

と述べた。

彼の理論は

```
difference
↓
pattern
```

であり、関係構造の分析である。

しかしここでも、**差異が持続する時間構造** は明示されない。

```
             保存系列
           (trace / time)

                 ↑
                 │
                 │
      現象学      │      ベルクソン
      Husserl    │      durée
                 │
                 │
──────────────更新──────────────
                 │
                 │
 構造主義         │      ベイトソン
 Saussure        │      pattern
 Lévi-Strauss    │
                 │
                 ↓

             空間系列
          (structure)
```

---

# 5｜更新存在論

更新存在論では、空間系列と保存系列は対立ではない。

両者は、**更新** から生成される。

```
update
↓
encounter
↓
trace
↓
persistence
↓
structure
```

ここで、保存系列は

```
trace
↓
memory
↓
history
```

であり、空間系列は

```
structure
```

として現れる。

```
           UPDATE
            │
            │
        encounter
            │
            │
           trace
            │
            │
        persistence
            │
            │
      ┌───────────────────┐
      │                   │
保存系列                   空間系列
trace → memory → history      structure
```
**Figure — Generation of persistence and structure**  
Update produces traces.  
Traces persist as memory and history.  
Structures emerge from persistent traces.  

Structuralism described spatial configuration.  
Phenomenology described temporal persistence.  
Updating ontology describes their common generation.

この視点から20世紀思想を再配置すると次のようになる。

|現象学|保存系列|
|ベルクソン|持続|

更新存在論は、**保存系列および空間系列から構造が生成される過程** を記述する。

# 結論

構造主義は空間系列を説明した。  
現象学は保存系列を説明した。

更新存在論は、**両者を生成する更新過程** を説明する。
