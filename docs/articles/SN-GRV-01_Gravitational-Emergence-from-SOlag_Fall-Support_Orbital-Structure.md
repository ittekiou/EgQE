### SN-GRV-01
# SO lagと重力
## ── 落下・支え・軌道構造の生成系列
### **Gravitational Emergence from SO lag — Fall, Support, and Orbital Structure**

---

# 0｜問い

なぜ重力は存在するのか。

既存物理では

- 質量がある
    
- 時空が曲がる
    

と説明される。

しかし、**なぜ落下が起きるのか** という生成の問いは残る。

EgQE構文では、重力を**lag構造の安定化運動**として読む。

---

# 1｜起源：SO lag

重力の起源も **SO lag** にある。

```
S（自己）
O（他者）
↓
非対称
↓
lag ≠ 0
```

lagは、**関係のズレ** である。

このズレは、**距離差** として現れる。

```
distance asymmetry
↓
lag gradient
```

これが、**落下の起源**になる。

---

# 2｜落下（fall）

lagが距離方向に偏ると、関係は**再配分**しようとする。

```
lag gradient
↓
fall
```

落下とは、**lag再配分の最小経路運動** 
である。

既存物理では

```
gravity
```

と呼ばれる。

---

# 3｜支え（support）

しかし宇宙には、完全な落下は存在しない。

常に

```
fall
×
support
```

が同時に存在する。

supportとは、**lagを局所的に固定する構造** である。

例

```
地面
星
惑星
```

---

# 4｜軌道形成

fallとsupportが共存すると、次の運動が生まれる。

```
fall
+
inertia
↓
orbit
```

軌道とは、**落下が支えによって曲げられた運動** である。

---

# 5｜構造形成

軌道が安定すると、重力構造が形成される。

```
fall
↓
orbit
↓
local structure
```

例

```
惑星系
銀河
星団
```

これは、**lagの空間的安定配置** である。

---

```

↓
structure
```

に対応する。

---

# 8｜帰結

重力とは、**lag勾配の空間的再配分** である。

落下は、lagを最小化する運動であり、軌道は、fallとsupportの平衡である。

---

# 結語

電磁力は、**衝突の蓄積** から生まれる。

重力は、**落下の持続** から生まれる。

宇宙構造とは、lagが

```
衝突
落下
```

という二つの運動として現れた結果である。
