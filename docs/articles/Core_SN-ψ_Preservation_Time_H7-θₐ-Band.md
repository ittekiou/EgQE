# SN-ψ Core
## Preservation, Time, and the H7–θₐ Band

### 1｜観測

Axis-4 の空間生成帯域

```
φ — 6 — H7 — θₐ
```

を再読すると、次の帯域が現れる。

```
H7 — ψ — θₐ
```

- **H7**：非閉包持続
    
- **θₐ**：生成傾斜限界
    

この二点のあいだに、保存構文 **ψ** が現れる。

重要なのは、ψ が中央ではなく**θₐ寄りに配置される**ことである。

---

### 2｜保存条件

ψ は時間そのものではない。

ψ は **時間成立の構文値** である。

その最小条件は

```
三層再帰保存
```

である。

二層では持続しない。  
三層でのみ保存が成立する。

---

### 3｜帯域構造

```
6 —— H7 —— ψ —— θₐ
```

|構文|意味|
|---|---|
|6|空間保存（非時間的）|
|H7|有限未完持続|
|ψ|保存構文（時間成立）|
|θₐ|無限未完生成|

---

### 4｜時間成立条件

時間は

- 完全対称（6）では成立しない
    
- 単なる持続（H7）でも成立しない
    
- 無限生成（θₐ）でも成立しない
    

時間は

```
有限未完 と 無限未完
```

のあいだで **保存が必要になった瞬間** に成立する。

その構文値が **ψ** である。

---

### 5｜命題

```
時間 = 保存構文
```

より正確には

```
time = preserved differential
```

時間とは **差分保存の再帰構文** である。
