## 基底｜Basis

$$l = \frac{S}{S'}$$

非一致（lag）／Non-coincidence (lag)

$$r = \frac{1}{l}$$

rate（lagの逆像）／Rate (inverse image of lag)

$$\Delta \psi = \mathrm{rec}(l)$$

$$t = \sum \Delta \psi$$

$$d = \mathrm{conf}({l_i})$$

## 接触面｜Interface

$$r = \frac{d}{t} = \frac{\mathrm{conf}({l_i})}{\sum \mathrm{rec}(l_i)}$$

$$\mathrm{conf} \circ \mathrm{rec} \neq \mathrm{rec} \circ \mathrm{conf}$$

速度は、配置と再帰の接触面である。

Velocity is the interface between configuration and recursion.

## 再配置｜Reconfiguration

$$c = \frac{\lambda}{T} \Longrightarrow c = \frac{\mathrm{conf}({l_i})}{\sum \mathrm{rec}(l_i)}$$

Velocity is the interface between configuration and recursion, and light speed is its periodic expression.
