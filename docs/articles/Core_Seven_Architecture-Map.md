# III｜Golden Domain & Heptagonal Hinge

### (Boundary Interaction Layer)

しかしその境界に、**Heptagonal Hinge** が現れる。

ここで重要なのは：

- Seven が対立を作るのではない
    
- Seven は吸収を停止させる最小ヒンジである
    

[数学強化版](https://camp-us.net/articles/TPD-02_Golden-Domain_Heptagonal-Hinge_Mathematical-Enhanced-Edition.html)では、このヒンジが **不可約回転粗視化条件**として定式化される。
