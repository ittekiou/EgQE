### # Note 1
# フッサールとメルロ＝ポンティ
## 論点整理（時間・身体・世界）

## 1　出発点：意識か身体か

### フッサール

出発点は **意識の構造**。

```text
現象 = 意識に現れるもの
```

現象学の基本命題：

> 意識は常に何かについての意識である  
> （intentionality）

つまり

```
主体（意識）
↓
志向
↓
対象
```

世界は **志向構造の中で現れる**。

---

### メルロ＝ポンティ

出発点は **身体**。

主体は純粋意識ではなく

```
身体
↓
知覚
↓
世界
```

主体は **身体化された存在（embodied subject）** になる。

---

# 2　時間

## フッサールの時間構造

有名な三層：

```
retention   過去保持
impression  現在
protention  未来予持
```

図にすると

```
過去保持
↓
現在
↓
未来予持
```

現在は点ではなく **厚みをもつ流れ**。

これは「内的時間意識」と呼ばれる。

---

## メルロ＝ポンティ

ポンティはこれを身体化する。

時間は意識の構造ではなく

```
身体の運動
↓
知覚の展開
↓
世界との関係
```

つまり **時間 = 生きられた時間**。

時間は

```
行為
知覚
身体
```

から生成される。

---

# 3　空間

### フッサール

空間は

```
志向の対象
```

つまり **認識構造の中の空間**

---

### メルロ＝ポンティ

空間は

```
身体が動ける場
```

つまり **行為空間**


例：

- 手が届く
    
- 歩ける
    
- 見える
    

これを **身体図式（body schema）** と呼ぶ。

---

# 4　世界

### フッサール

世界は

```
意識の地平
```

世界は常に **志向の地平として現れる**

---

### メルロ＝ポンティ

世界は

```
身体と絡み合う場
```

主体と世界は **絡み合い（chiasm）** になる。

---

### # Note 3

# ベイトソンはなぜ時間論を書かなかったのか

**① 書かなかった（必要がなかった）**  
**② 書けなかった（理論構造上の限界）**

実態は **その両方**。

---

# 1 書かなかった理由（方法論）

Gregory Bateson の理論の中心はこれ。

> information is a difference that makes a difference

つまり彼の世界は

```
difference
↓
relation
↓
pattern
↓
mind
```

ここでは **時間は主題にならない**。  

なぜなら、**パターンは時間をまたぐ構造だから**。

例えばベイトソンの代表概念：

- ダブルバインド
    
- 学習レベル (Learning I, II, III)
    
- サイバネティクス
    
- フィードバック
    

これらはすべて

```
interaction pattern
```

の理論。

つまり彼の関心は **「時間の流れ」ではなく「関係の循環」**。

---

# 2 サイバネティクスの時間

ベイトソンが属していた知的環境は

- サイバネティクス
    
- 情報理論
    
- システム理論
    

この世界の時間は

```
feedback loop
```

つまり

```
A → B → C → A
```

という **循環時間**。

この枠組みでは **持続（durée）や生成時間は出てこない。**

だから、時間論は主題にならなかった。

---

# 3 書けなかった理由もある

ベイトソン理論の基礎は

```
difference
```

しかし彼は、**differenceがどこから来るか** を説明していない。

つまり

```
difference
↓
pattern
↓
mind
```

は書いたが

```
difference
↑
origin
```

を書いていない。

---

# 4 そこに時間がある

実は、**differenceの起源を問うと時間が出てくる。**

なぜなら、differenceとは

```
before ≠ after
```

過去と未来の非対称。

つまり

```
difference
＝ temporal asymmetry
```

ベイトソンは、この「生成」を扱わず、**関係のパターン**を扱った。

---

# 5 ベルクソンとの違い

Henri Bergson

```
durée
↓
difference
↓
life
```

ベイトソン

```
difference
↓

↓
mind
```

つまり、**differenceの起点が違う。**

ベルクソン → 持続

ベイトソン → 情報


**ベイトソンは時間論を書かなかった。  
なぜなら彼の理論は関係パターンを扱う理論だったから。**

しかし同時に

**differenceの生成原理を扱わなかったため、時間論を書けなかったとも言える。**

構造主義は基本的に


- 言語構造




```id="78m4b7"

↓
```


Ferdinand de Saussure
