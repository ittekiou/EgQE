# Prologue｜火・舌・文字──R₀⇄Z₀変換史の起点

**火が脳を育て、舌が音を切り、  文字が世界をZUREへと変えた。**  
ここから、ホモ・サピエンスの R₀⇄Z₀ 変換史が始まる。

人類の歴史は、  **R₀（連続する世界）を  Z₀（切断された構文）へ書きかえる物語であった。**

連続はゆらぎであり、切断は痕跡であり、  その両者を往復するのが **言語**であり、  とりわけ **Articulation と Script** である。

ホモ・サピエンスは、  音声によって世界の連続を「切り取り」、  文字によってその切断を「固定し」、  さらに制度・科学・文明へと拡大した。

こうして、  **R₀ → Z₀ → R₀**  という巨大な往還運動の歴史が始まった。

# 第一部：ホモ・サピエンス編  
## ── R₀世界からZ₀構文化への道

ここでは、生物史・感覚史・記号史を  **R₀（analog / continuous）→ Z₀（digital / discrete）** の軸で再構成する。

## 第1章：R₀世界の誕生 ── 五感と火の革命

### 1. 火と調理がもたらした「余剰エネルギー」
- 調理が消化の負担を軽減し、脳へ大量のATP供給を可能にした  
- 「余白（free energy）」が生まれ、脳容量が増大

### 2. 脳の跳躍
- 舌運動の精密化  
- 聴覚の周波数分解能向上  
- 前頭前野の拡張  
- 社会的コミュニケーションの深化

### 3. 手・道具・社会の三位一体
- 手：世界への“切れ目”  
- 道具：構造化  
- 社会：再利用  
- この段階からすでに **R₀世界のZURE化** が始まっていた

## 第2章：Articulationの誕生 ── 聴覚R₀に刻まれる最初のZURE

### 1. Articulation は聴覚起源
- 音＝連続（R₀）
- 音素＝切断（ZURE）  
- これが **Z₀前段階** の成立

### 2. 舌・口蓋の発達と発話運動
- 火と調理が舌・口蓋の微細運動を進化させた  
- 舌→音の切断→ articulation → phoneme

### 3. phoneme = 世界の最小ZURE単位
- 自然界の音は連続  
- 言語音は離散  
- **R₀ → ZURE → Z₀** の最初の回路がここで成立

---

## 第3章：Scriptの出現 ── 視覚Z₀による外部記憶の始動

### 1. 文字は “音のZURE” を視覚へ転写
- 音：消えるZURE  
- 文字：残るZURE（永続化）

### 2. 書記体系は Z₀構文化の決定的瞬間
- 視覚化 → 離散化 → 再利用 → 抽象 → 記号論

ここで **Z₀が人類史上初めて完全成立**。

---

## 第4章：外部脳としての文明 ── 言語・数式・科学の誕生

### 1. 神話 → 文法 → ロジック → 数学
ZUREの段階的抽象化による文明化。

### 2. Z₀脳 が R₀脳 を包み込む二層化
- R₀：身体・感覚  
- Z₀：構文・抽象  
- 二層構造により文明が立ち上がる
