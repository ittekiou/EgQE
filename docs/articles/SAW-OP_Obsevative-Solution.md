# O. 零点構文の **非可換代数**

## 命題 1｜零点構文は可換ではない

> lag に対して「何を・どの順で固定するか」

    と

は、**異なる履歴**を生成する。

#### 結論

零点構文は代数的に：

> **非可換演算子**

これらは一般に：

> **Zₓ Zₚ ≠ Zₚ Zₓ**

❌ 量子量が非可換 ではない。

✅ **零点構文が非可換**

量子力学の非可換性は、**世界の性質ではなく、観測零点構文の性質**。
