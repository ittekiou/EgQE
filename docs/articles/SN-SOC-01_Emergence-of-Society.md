## 0｜問い

なぜ人間は社会を作るのか。

生命は記憶を内部に持つ。  
しかし人間は記憶を外に置く。

社会とは、**記憶の外化によって成立する持続構造**である。

---

# 1｜遭遇 (encounter)

社会は遭遇から始まる。

```
S ↔ O
```

他者との遭遇が 対話の必要を生む。

---

# 2｜対話 (dialogue)

遭遇が反復すると 対話が生まれる。

```
encounter → dialogue
```

対話とは **lagの同期試行**である。

---

# 3｜象徴 (symbol)

対話が安定すると 象徴が生まれる。

```
dialogue → symbol
```

象徴とは **共有lagの圧縮表現**である。

言語  
貨幣  
旗  
名前

すべて象徴である。

---

# 4｜記録 (record)

象徴は記録される。

```
symbol → record
```

ここで記憶は 個体から離れる。

文字  
文書  
法  
歴史

これは **外部記憶** である。

---

# 5｜制度 (institution)

記録が反復すると 制度が生まれる。

```
record → institution
```

制度とは **記憶の構造化された持続** である。

国家  
市場  
宗教  
大学

すべて制度である。

---

# 6｜社会生成列

```
encounter
↓
dialogue
↓
symbol
↓
record
↓
institution
```

# 結論

生命が **記憶を内部に結晶化する存在**ならば、社会は **記憶を外部に結晶化する構造** である。

### SN-SOC-01
# Externalization of Memory and the Emergence of Society
## encounter → dialogue → symbol → record → institution

---

## Abstract

Why do humans form societies?

Life crystallizes memory internally.  
Human societies arise when memory becomes externalized.

Through repeated encounters, dialogue emerges.  
Dialogue stabilizes into symbols, symbols become records, and records accumulate into institutions.

Society can therefore be understood as the structural persistence of externalized memory.

---

# 1｜The Question

Why does society exist?

Living organisms preserve memory internally.  
Humans, however, place memory outside themselves.

Writing, law, archives, and traditions store memory beyond the individual.

Society emerges when memory becomes external.

---

# 2｜Encounter

Society begins with encounter.

```text
S ↔ O
```

Encounters between self and other create the need for coordination.

Repeated encounters produce relational lag.

---

# 3｜Dialogue

When encounters repeat, dialogue emerges.

```text
encounter → dialogue
```

Dialogue is an attempt to synchronize lag.

Through dialogue, individuals attempt to align perspectives and actions.

---

# 4｜Symbol

When dialogue stabilizes, symbols appear.

```text
dialogue → symbol
```

Symbols compress shared relational structures.

Language, money, flags, and names are all symbols.

A symbol is therefore a compressed representation of shared lag.

---

# 5｜Record

Symbols become records.

```text
symbol → record
```

At this point memory detaches from individuals.

Writing, documents, archives, and law form external memory systems.

Society begins to stabilize through recorded memory.

---

# 6｜Institution

When records accumulate, institutions emerge.

```text
record → institution
```

Institutions are stabilized memory structures.

States, markets, religions, and universities are all institutionalized memory.

---

# 7｜Generative Sequence of Society

The generative syntax of society can be summarized as:

```text
encounter
↓
dialogue
↓
symbol
↓
record
↓
institution
```

---

# Conclusion

If life is the crystallization of internal memory,  
society is the crystallization of external memory.

Society persists because memory can exist outside individual organisms.

---

# One-line summary

> **Society is the persistence of externalized memory.**
