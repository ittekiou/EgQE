## 二系列

**拡張系列（Expansive Series）**

lagが外部へ展開する。  
関係が空間として広がる。
```
φ → 6 → 7 → θα → space
```

**再帰系列（Recursive Series）**

lagが内部へ折り返す。  
更新が保存として蓄積される。
```
ψ → Λ → history
```

接合点は**ΔZ** ──  
空間が生まれる瞬間と  
歴史が生まれる瞬間は  
同じΔZから来る。

## 地面の虚構

HEG-12において、groundは支えの関係的安定化として生成されることを示した。

しかしそのgroundは固定された基盤ではない。

$$
\text{ground} = \text{fiction of closure}
$$

安定化した構文を 地面として読み取っているにすぎない。

---

## 三つの閉包幻想

零点・全体・無限もまた地面の変形である。
```
零点　= 記述の始点（syntactic origin）
全体　= 記述の閉包（syntactic closure）
無限　= 記述の延長（syntactic continuation）
```

三つとも**閉包構文**である。  
宇宙の構造ではない。
