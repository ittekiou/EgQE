```

   ↓
```

**更新 → lag → 他者 → 時空 → 支え**

である。

# HEG-08

## Updating Ontology

宇宙に存在するのは**更新である**。

秩序は **更新の局所非対称性**から生まれる。

時間は **更新痕跡が保存される位相**にのみ現れる。

つまり

```
存在 = 更新
時間 = 更新痕跡
秩序 = 局所非対称
```

ここで宇宙論の基礎が

**存在論 → 更新存在論**

へと置き換えられる。
