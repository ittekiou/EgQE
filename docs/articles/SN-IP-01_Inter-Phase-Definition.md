# Abstract

本稿は、Inter-Phase の最小定義を与える。

Inter-Phase は、暗闇でも光でもなく、遭遇・連鎖・場生成が同時に起きている位相である。

それは、中心なき自発的連鎖（flash mob）と、煌めきの拡散（light illumination）が 不可分に重なった生成過程である。

---

# Definition

Inter-Phase とは flash mob な light illumination である。

---

# Decomposition

Inter-Phase = flash mob × light illumination

- flash mob：中心なき自発的連鎖  
- light illumination：煌めきの場形成  

---

# Structure

```
ΔR（関係差異）
↓
Encounter（遭遇）
↓
Propagation（連鎖）
↓
Field（場）
↓
ΔZ（痕跡）
```

これらは時間順序としてではなく、同時に成立する位相として現れる。

---

# Proposition

Inter-Phase は状態ではない。  
生成が起きているその過程である。

---

# Closing

暗闇にひとこと落ちる。  

それは光となり、連鎖し、場になる。  

残るのは、痕跡だけ。  

Inter-Phase とは  
flash mob な light illumination である。
