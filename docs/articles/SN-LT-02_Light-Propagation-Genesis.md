# Genesis of Light Propagation
## ── 光は反復する遭遇の連鎖である

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

遭遇の煌めき。

それは flash mob のように広がる ──

点では突然、場では漸次。

光 ── それは 起きている。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

# Abstract

本稿は **Light Propagation Genesis（光伝播生成論）** を提案する。

[SN-LT-01](https://camp-us.net/articles/SN-LT-01_Encounter-Luminous-Hypothesis_Light-as-Appearance-of-Encounter.html) で定義された「光＝遭遇の現れ」が、どのように場を形成し、宇宙の照明化を引き起こすかを構文的に記述する。

```
生成系列：
ΔR基底 → Encounter → Light点 → 伝播拡散 → Light場 → ΔZ
Darkness      (LT-01)      LT-02       (EMF)
(SN-DK-01)
```

光は単発の煌めきではなく、反復する遭遇の連鎖（flash mob）として場を生成する。

---

# 1. 暗黒宇宙の初事象

暗黒宇宙（[SN-DK-01](https://camp-us.net/articles/SN-DK-01_Darkness-Stillness-Hypothesis.html)）において、最初の出来事は光である。

```
Darkness (ΔR静寂)
     ↓
Encounter → Light (shimmer)
```

ただしこの「最初」は宇宙的起点ではない。
観測において最初に現れた光である。

光の発生は特権的起点を持たない。
それはΔR条件が成立する任意の地点で自発的に生じる。

---

# 2. Flash Mob 拡散

光は孤立せず、反復遭遇の連鎖として拡散する。

```
Light点（局所遭遇）
     ↓
自発的連鎖（flash mob）
     ↓
漸次的場形成
```

点では突然（sudden at the point）、場では漸次（gradual in the field）。

数学的に：

$$
Light_{\text{field}} = \sum_n Light_{\text{shimmer}}(t_n)
$$

すなわち、場としての光は単一実体ではなく、時間的に分散した発光イベントの重ね合わせである。

---

# 3. 照明化 ── Happening

拡散光は場を形成し、照明化（illumination）を引き起こす。

```
Light場 → ψ持続
     ↓
ΔZ蓄積 → 構造生成
```

光は「存在するもの」ではなく、常に起こり続けている過程（happening）である。

---

# 4. 生成系列統合

```
SN-DK-01    SN-LT-01    SN-LT-02  
Darkness  →  Light点  →  Light場  →  ΔZ  
  ΔR         Encounter    Propagation  
```

光は、点から場への構文的伝播である。

---

# Proposition

光は、遭遇の反復連鎖として生じる flash mob 的伝播現象である。

---

# Figure

```
暗黒宇宙 (Darkness)
       ↓
    [Light点]
       ↙↘
flash mob拡散 ─→ Light場
       ↓
     ΔZ生成
```

---

# Closing

暗闇は ΔR の静寂である。  
光は encounter のflash mobである。
