### SP-07｜Syntactic Value
# 構文価値論 序説
# Toward a Theory of Syntactic Value
## — From Fixation to Circulation —

---

## 0. 導入｜問いの反転

### 0. Introduction｜Reversing the Premise

価値はどこから来るのか。  
労働か、効用か、交換か。

しかし問うべきはこれである：

**価値はどのように"固定され"、いかに"流通する"のか。**

Where does value come from?  
Labor? Utility? Exchange?

But the question is:

**How is value fixed — and how does it circulate?**

---

## 1. 価値の再定義｜Value as Fixation

### 1. Redefining Value

EgQEにおいて：

```
Z₀ = 差異の切断（event）
ΔZ = 切断の痕跡（fixation）
```

本稿の定義：

```
構文価値 = ΔZ
         = 交換可能な形に固定された差異
```

> **価値とは、差異が交換可能な構文として固定されたものである。**  
> **Value is difference fixed as an exchangeable syntax.**

---

## 2. 数値化と安心｜From Number to Assurance

### 2. Numericization and Stability

```
差異
↓
数値化
↓
固定（ΔZ）
↓
安心
```

数値化は記述ではない。  
**固定の技術である。**

数値は、差異を可搬な形に固定する。

Numericization is not description.  
**It is the technology of fixation.**

Numbers fix difference into portable form.

---

## 3. 交換と制度｜Circulation

### 3. Exchange and Institution

```
固定（ΔZ）
↓
交換可能
↓
価格化
↓
制度化
```

価値は内在しない。  
流通条件の中で機能する。

> **価値とは、固定が流通に入ったときに立ち上がる。**  
> **Value emerges when fixation enters circulation.**

---

## 4. 鹸化としての市場

### 4. Market as Saponification

```
直接対面
↓
間接化
↓
制度
```

市場とは：

```
非混和（他者）
↓
鹸化（条件の再配置）
↓
接続可能性
```

> **市場は、関係を成立させる条件を再編する鹸化装置である。**  
> **The market is a saponification device that reconfigures the conditions of relation.**

---

## 5. 自立と疎外｜Autonomy and Alienation

### 5. Autonomy and Alienation

```
構文片（ΔZ）
↓
関係から切断
↓
自立
```

自立とは自由ではない。  
**関係からの距離化である。**

> **疎外とは、構文が関係から切り離されて自立した状態である。**  
> **Alienation is the state in which syntax has detached from relation and become autonomous.**

---

## 6. 三つの価値

### 6. Three Kinds of Value

```
語用価値 = 使われ方
意味価値 = 内容
構文価値 = 固定と流通
```

本稿の主張：

> **構文価値は、他の価値を流通可能にする基盤である。**  
> **Syntactic value is the substrate that makes other values circulable.**

---

## 7. 一撃命題

### 7. Core Propositions

> **価値は内在しない。固定され、流通する。**  
> **Value does not inhere; it is fixed and circulates.**

> **排泄されたものだけが市場に出る。**  
> **Only what is excreted enters circulation.**

---

## 結語｜Conclusion

価値は発見されるのではない。  
生成される。

それは差異の固定であり、その固定が流通することで機能する。

> **価値とは、排泄された差異の循環である。**  
> **Value is the circulation of excreted difference.**

---
