# 価値の争いからバイアスの選択へ
### ── 拘りから自由な政治学へ ②

政治は長いあいだ、価値の争いとして語られてきた。

自由か平等か。

市場か国家か。

保守か革新か。

価値は重要である。

しかし、価値だけでは制度は設計できない。

制度は必ず、何らかのバイアスを持つ。

制度とは、ある方向を選び、ある方向を選ばない仕組みだからである。

だから制度改革とは、価値を争うことではない。

制度目的を確認し、その目的に最も適したバイアスを選択することである。

ここには唯一の正解はない。

同じ制度目的であっても、現実の条件が違えば、選ばれるバイアスも変わる。

だから制度には相対性がある。

しかし、それは相対主義ではない。

制度目的は共有できる。

相対的なのは、その目的へ至る制度設計である。

政治とは、その選択を引き受ける営みである。

問題はバイアスではない。

バイアスを唯一の正しさとして固定する拘りである。

拘りは制度目的と制度設計を混同する。

制度は未来のために存在する。

しかし拘りは、自らを守るために制度を利用する。

そこで制度改革は止まる。

制度改革とは、拘りを別の拘りへ置き換えることではない。

制度目的を見失わず、複数のバイアスを露出させ、現実に即して未来のためのバイアスを選び続けることである。

政治とは、価値を叫ぶことではない。

未来を支える制度を編集することである。

そのために必要なのは、価値への忠誠ではなく、拘りから自由な思考である。

未来とは、拘りの先にあるのではない。

未来とは、選び直すことのできるバイアスの中にある。
