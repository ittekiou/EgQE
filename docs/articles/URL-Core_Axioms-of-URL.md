# URL-Core
## ── Axioms of URL ──

> Lag is not eliminated. It is displaced.

---

### 0｜Axioms

```
difference = relation
ratio      = expression of relation
lag        = non-coincidence within ratio
rate       = unfolding of ratio
```

```
差異 = 関係
比   = 関係の表現
lag  = 比の非一致
rate = 比の展開
```

---

### 1｜Thesis

比が展開し  
揃わぬがゆえ  
世界は立ち上がる

---

Ratio unfolds,  
does not coincide,  
the world emerges.
