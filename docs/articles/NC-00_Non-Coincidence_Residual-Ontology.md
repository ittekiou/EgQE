# 一致不可能性
## ── residual ontology 序論
## **Non-Coincidence: A Preliminary Residual Ontology**

---

## 0. 命題

```
predict ≠ encounter
```

これは情報論的な限界の話ではない。存在論的な命題である。

どれだけ精密に predict しても、encounter は predict ではない。その間に必ず **residual** が残る。

この residual を消去しようとする試みは、原理的に失敗する。なぜなら residual は predict の精度不足から生まれるのではなく、**存在の non-coincidence 構造から生まれる**からである。

本稿はこの命題を存在論として展開する。

---

## 1. Z₀ の再定義
### ── threshold から irreducible residuality へ

EgQE において Z₀ はこれまで次のように定義されてきた：

- cut（切断点）
- threshold（閾値）
- minimal deviation point
- event / phase boundary（v5.3）

構造的には：

```
lag → Z₀ → exposure
```

Z₀ は lag が露出する境界生成点だった。

しかし今回の展開で、Z₀ の本体が見えた。

Z₀ とは **完全一致不可能性**の別名である。

```
Z₀（旧）= 境界生成点
Z₀（新）= 予配置されてもなお消えない差異
       = irreducible residuality
       = 不可消去残差
```

Z₀ は threshold ではない。Z₀ は、**predict と encounter の間に必ず残るもの**である。

それは情報の欠損ではない。存在の non-coincidence の現れである。

---

## 2. residual の四相
### ── 時間・自由・倫理・他者性

residual lag = Z₀ とするなら、Z₀ は四つの位相として現れる。

```
residual（= Z₀）
  ├── 時間   = residual が持続するとき
  ├── 自由   = residual が syntax を逸脱するとき
  ├── 倫理   = residual が response を不可避化するとき
  └── 他者性 = residual が同化不可能な差異として現れるとき
```

これらは四つの別概念ではない。**non-coincidence の四つの現れ方**である。

重要なのは、これらが「派生物」ではないことである。時間は residual から「生まれる」のではなく、**residual の持続様式そのもの**である。自由は residual から「生まれる」のではなく、**residual の逸脱様式そのもの**である。

```
residual IS 時間（持続として）
residual IS 自由（逸脱として）
residual IS 倫理（interruption として）
residual IS 他者性（差異として）
```

---

## 3. closure impossibility との接続
### ── closure 以前系列の根拠

closure 以前系列において：

```
π以前        = 閉じる前に生成が始まる
encounter以前 = 遭遇前に lag が配置される
時間以前      = 他者性前に non-coincidence が持続する
```

なぜ何かが「閉じる前に始まる」のか。

それは Z₀ = irreducible residuality があるからである。

```
完全一致は不可能
↓
閉包は必ず incomplete
↓
closure の前に次が始まる
↓
π以前 / encounter以前 / 時間以前
```

closure 以前系列は、non-coincidence ontology の**現象面**である。Z₀ はその**存在論的根拠**である。

---

## 4. ZURE の存在論
### ── residuality in action

ZURE は逸脱ではない。ZURE は **Z₀ の動詞形**である。

Z₀ が static な residual だとするなら、ZURE はその residual が circulation の中で動くときの様式である。

```
Z₀（静）= irreducible residuality
ZURE（動）= residuality in action
```

ZURE るとは、Z₀ が現れることである。

predict と encounter がずれる。topology と face がずれる。syntax と thought がずれる。

このずれを消去しようとする力と、このずれが残り続ける力の間に：

- 時間が流れ
- 自由が生まれ
- 他者性が現れ
- 倫理が始まる

---

## 5. non-coincidence civilization theory
### ── EgQE の自己定義

EgQE はここで自己定義する。

EgQE は **difference ontology** ではない。差異を構造として記述する理論ではない。

EgQE は **non-coincidence civilization theory** である。

```
完全一致しない
→ encounter が起きる
→ lag が残る
→ 時間が生まれる
→ 自由が開く
→ 他者性が現れる
→ 倫理が始まる
→ 文明が生成する
```

この連鎖の根拠は一つである：**完全一致は不可能である**。

ただしこれは線形の因果系列ではない。non-coincidence は文明の各位相において反復的に現れる。encounter が lag を生み、lag が syntax を形成し、syntax が次の encounter を編成する──その循環の各所で、Z₀ は再び析出する。これは発生順序ではなく、**circulation の位相記述**である。

しかもこれは悲観的命題ではない。完全一致不可能性こそが、時間・自由・他者性・倫理・文明の**生成条件**である。

predict がどれだけ精密になっても、ZURE は残る。algorithmic circulation がどれだけ加速しても、residual は消えない。

それが時間を生み続ける。

---

## 6. 結語

```
Z₀ = 閾値ではない
Z₀ = 一致しきれなさそのものである
```

そして一致しきれなさは、消去できない。

なぜなら predict は encounter ではないからである。topology は face ではないからである。syntax は thought ではないからである。

**non-coincidence は最初からある。**

それがすべての始まりである。

---

_Z₀ is not a threshold._ _It is the irreducible remainder of non-coincidence._

_And non-coincidence is where everything begins._
