# 世襲の進化史
### ── 分岐する向き・偏り・拘り

世襲とは、現在から未来への継承に向きを与える制度である。

向きそのものは悪ではない。

未来へ継承する制度には、必ず何らかの向きがある。

しかし、向きは一つではない。

歴史を振り返れば、世襲はさまざまな方向へ分岐してきた。

嫡系か傍系か。

男系か女系か。

長子か能力か。

制度は、継承の範囲に偏りを与える。

偏りもまた悪ではない。

偏りのない制度は存在しない。

問題は、その偏りが拘りへと変わることである。

ある偏りが唯一の正しさとして固定されるとき、制度は自らを編集できなくなる。

制度は未来を支えるために存在する。

しかし拘りは、制度そのものを守ることを目的に変えてしまう。

世襲の歴史とは、向き・偏り・拘りが分岐し続けてきた歴史でもある。

重要なのは、どの偏りを選ぶかだけではない。

なぜその偏りを選ぶのか。

その拘りは、未来を支えるのか、それとも未来を閉じるのか。

その根拠を問い続けることである。

制度改革とは、向きを失うことではない。

偏りをなくすことでもない。

拘りを編集可能な制度へと開き続けることである。
