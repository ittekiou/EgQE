## 1｜要点（5）

1. **測り方の転換**：E(k) の「−5/3 直線」探しではなく、**小スケール全域**での**崩壊**を検定。→ **小スケール普遍性**が明瞭化。  
2. **TC乱流は“例外”ではない**：従来「慣性域が見えにくい」ため非普遍とされたが、**崩壊**では**普遍曲線に綺麗に重なる**。  
3. **a=1**（**等角速度の反対回転**）では従来どおり **−5/3** が出現。ただし**崩壊の成否と −5/3 の可視性は別問題**と整理。  
4. **λ（テイラー微小スケールRe）で整列**：崩壊域の広がりは **λ** の増大に**単調対応**。位相空間（回転比 *a*, Re）横断で一貫。  
5. **新たな示唆**：崩壊から外れ始める**より大スケール側**で**−2/3**らしきスケーリング域を示唆（要追試）。
